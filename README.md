# Brain-Tumor-AI

Project scaffold for brain tumor detection and classification with explainable AI.

## Structure

- `data/` for raw, processed, and augmented datasets
- `notebooks/` for EDA, preprocessing, training, comparison, and Grad-CAM analysis
- `models/` for saved Keras models
- `src/` for reusable Python modules
- `app/` for the Streamlit app
- `outputs/` for plots, reports, confusion matrices, and heatmaps

## Notes

Place your dataset in `data/raw/` or adapt the pipeline to point at the existing `Training/` and `Testing/` folders in the workspace.
